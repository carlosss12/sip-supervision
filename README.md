# Sistema de Supervisión - S.I. Protection

Despliegue automatizado del proyecto mediante Docker Compose.

## Instrucciones de Despliegue

Realizará estos tres pasos:

1. Clonar el repositorio:
   git clone https://github.com/carlosss12/sip-supervision-docker.git

2. Ingresar al directorio:
cd sip-supervision-docker

3. Levantar el proyecto:
docker compose up
